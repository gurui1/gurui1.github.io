websocket

发布时间 : 2024-04-19 21:36
阅读 :

这段代码确实建立了一个 WebSocket 长连接，并在连接建立后发送了初始化消息。让我来解释一下：

  1. WebSocket 连接建立：

    url: “ammoniaGas/get”,不需要写 因为WebSocket 长连接不需要像传统的 HTTP 请求那样指定接口路径。
    这一行代码通过 new WebSocket() 创建了一个 WebSocket 对象，并指定了服务器的地址和端口号。这样就建立了与服务器的 WebSocket 连接。

  2. 连接建立后发送初始化消息：

    当 WebSocket 连接成功建立时，ws.onopen 事件触发，会执行这个回调函数。在这个回调函数中，客户端发送了一个初始化消息，通过 ws.send() 方法发送给服务器。这个初始化消息包含了订阅动作 (action: "subscribe") 和订阅的主题 (topic: "AEData")，以及附带的参数 (payload)。

  3. 处理服务器发送的消息：

    当客户端接收到来自服务器的消息时，ws.onmessage 事件触发，会执行这个回调函数。在这个回调函数中，客户端处理接收到的消息，更新界面或执行其他操作。在你提供的代码中，处理了接收到的数据，并更新了图表。
