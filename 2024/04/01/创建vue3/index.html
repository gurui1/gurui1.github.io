1 新建项目

新建文件夹 win+r cd进入
输入 npm create vite@latest
输入 项目名 选择 vue ts
cd进入项目 cnpm i 下载依赖
npm run dev 运行

2 安装vue-router

vue3需要安装4.0以上版本
vue2最好安装4.0以下版本

2.1 安装路由命令

npm install vue-router@next –save // 安装最新版本router

// 如需按版本安装，需将命令行中 next 改成相应的版本。如下：
// npm install vue-router@4.0.13 –save

安装完成后，可在package.json中查看vue-router是否安装完成

2.2 配置router文件

src目录下新建一个routes文件夹，在router文件夹里新建一个index.ts文件，代码如下:

import { createRouter, createWebHistory, RouteRecordRaw } from “vue-router”;

// 1. 配置路由
const routes: Array = [
{
path: “/“, // 默认路由 home页面
component: () => import(“../views/home/index.vue”),
},

];
// 2.返回一个 router 实列，为函数，配置 history 模式
const router = createRouter({
history: createWebHistory(),
routes,
});

// 3.导出路由 去 main.ts 注册 router.ts
export default router

2.3 在main.ts中引用router下的index.ts 进行配置

import { createApp } from ‘vue’ // 安装unplugin-auto-import 可注释
import App from ‘./App.vue’
import router from “./router/index” // 引入router

const app = createApp(App)
app.use(router)
app.mount(‘#app’)

2.4 app.vue中添加路由组件router-view

在div里面插入router-view
