数组方法

前端
发布时间 : 2024-03-17 17:02
阅读 :
  1. 添加 / 删除元素：
  2. push()：向数组末尾添加一个或多个元素，并返回新的长度。
  3. pop()：删除数组最后一个元素，并返回该元素的值。
  4. shift()：删除数组的第一个元素，并返回该元素的值。
  5. unshift()：向数组的开头添加一个或多个元素，并返回新的长度。
  6. 连接 / 分割数组：
  7. concat()：连接两个或多个数组，并返回一个新数组。
  8. slice()：从已有的数组中返回选定的元素。slice() 方法会返回一个新数组，该数组包含从原数组中提取的元素，包括起始索引，但不包括结束索引。
  9. 修改数组：
  10. splice()：从数组中添加 / 删除元素。
  11. 迭代数组：
  12. forEach()：对数组的每个元素执行一次提供的函数。
  13. map()：创建一个新数组，其结果是该数组中的每个元素调用一个提供的函数后返回的结果。
  14. 颠倒元素顺序：
  15. reverse()：颠倒数组中元素的顺序。

添加 / 删除元素：

push()：向数组末尾添加一个或多个元素，并返回新的长度。

let arr = [1, 2, 3];
console.log(arr.push(4));## 4
console.log(arr);## [1, 2, 3, 4]

pop()：删除数组最后一个元素，并返回该元素的值。

arr.pop(); ## [1, 2, 3]

shift()：删除数组的第一个元素，并返回该元素的值。

arr.shift(); ## [2, 3]

unshift()：向数组的开头添加一个或多个元素，并返回新的长度。

arr.unshift(22,11);
console.log(arr);## [22,11, 2, 3]

连接 / 分割数组：

concat()：连接两个或多个数组，并返回一个新数组。

slice()：从已有的数组中返回选定的元素。slice() 方法会返回一个新数组，该数组包含从原数组中提取的元素，包括起始索引，但不包括结束索引。

let arr1 = [1, 2];
let arr2 = [3, 4];
let mergedArr = arr1.concat(arr2); ## [1, 2, 3, 4]
let slicedArr = mergedArr.slice(1, 3); ## [2, 3]

修改数组：

splice()：从数组中添加 / 删除元素。

let arr3 = [1, 2, 3, 4, 5];
arr3.splice(2, 2); ## 从索引为2的位置删除2个元素
console.log(arr3);
arr3 现在为 [1, 2, 5]

迭代数组：

forEach()：对数组的每个元素执行一次提供的函数。

map()：创建一个新数组，其结果是该数组中的每个元素调用一个提供的函数后返回的结果。

let arr4 = [1, 2, 3, 4];
arr4.forEach(element => console.log(element)); ## 输出每个元素
let doubledArr = arr4.map(num => num * 2); ## [2, 4, 6, 8]

颠倒元素顺序：

reverse()：颠倒数组中元素的顺序。

let arr5 = [1, 2, 3, 4, 5];
arr5.reverse();
现在 arr 变为 [5, 4, 3, 2, 1]
